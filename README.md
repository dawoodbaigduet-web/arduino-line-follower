# 🤖 Arduino Line Follower Robot

## 📌 Overview
This project is an Arduino Nano based Line Follower Robot developed during my Electronics Engineering studies at Dawood University of Engineering and Technology (DUET).

## 🔧 Components Used
- Arduino Nano
- IR Sensor Module
- L298N Motor Driver
- DC Motors
- Robot Chassis
- Battery Pack

## ✨ Features
- Autonomous line tracking
- Black line detection
- Motor speed control
- C++ programming
- Embedded Systems
